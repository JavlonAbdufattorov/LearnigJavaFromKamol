# Scanner,print и println в Java

---

## Scanner — ввод данных с клавиатуры

`Scanner` — это **класс**, который используется для ввода данных от пользователя.

Находится в пакете:
```java
java.util.Scanner
```
Для подключения класса `Scanner` необходимо импортировать пакет,
который был упомянут ранее.

---
Далее надо создать обект Scanner
```java
Scanner scanner = new Scanner(System.in);
```
`System.in` — стандартный поток ввода (клавиатура)

---
Есть разные типы ввода данных

### Ввод целого числа
```java
int a = scanner.nextInt();
```
### Ввод дробного числа
```java
double b = scanner.nextDouble();
```
### Ввод одного слово до пробела
```java
String word = scaner.nextDouble();
```
### После `nextInt()`, `nextDouble()`, и т.п перед `nextLine()` нужно съесть перевод строки:





Методы `Scanner` **по-разному читают ввод**:

- `nextInt()`, `nextDouble()`, `next()`
    читают **только значение**
- `nextLine()`
    читает **всю строку до конца**, включая нажатие `Enter`

Когда пользователь нажимает `Enter`, в потоке ввода появляется **символ новой строки**:
\n

---
## Что реально вводит пользователь

```java

Scanner scanner = new Scanner(System.in);

int age = scanner.nextInt();
String name = scanner.nextLine();

System.out.println("Имя: " + name);
```

Пользователь вводит: 25

На самом деле в поток попадает:25\n

`nextInt()` читает 25 не читает \n оно остается в буфере

`nextLine()` сразу читает оставшийся \n считает что строка пустая и пользователь не успевает еичего ввести.
